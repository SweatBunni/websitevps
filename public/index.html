Capabilities

Everything you need
to mod Minecraft

CodexMC understands the full Minecraft Java modding ecosystem — from event systems to build configs.

Complete Java Code
Fully runnable Java with all imports, package declarations, and class structure — ready to compile with zero edits needed.
Full Gradle Projects
Outputs build.gradle, gradle.properties, settings.gradle with correct loader configs, Yarn mappings, and API versions.
Live Version Research
Fetches current Yarn mappings, Fabric API, and ForgeGradle versions from official sources before generating code.
Resource Files Included
Model JSON, texture references, fabric.mod.json or mods.toml — every resource file your mod needs is generated automatically.
One-Click JAR Build
Hit "Build Mod" and CodexMC compiles using real Gradle on the server. AI auto-fixes errors. Download the working JAR.
Streaming + Activity Log
Real-time streaming responses with a live console showing each step — from research to code generation to build completion.
